### 4. What will display in the web browser tab for the following HTML document?

```
<!DOCTYPE html>
<html>
<head>
    <title>Little Lemon</title>
</head>
<body>
    <p>Our Menu</p>
</body>
</html>
```

- [x] Little Lemon. <br>
      Well done. The content inside the title element will display as the web page name in the browser tab.
- [ ] Our Menu.
